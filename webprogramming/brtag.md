## 줄바꿈`<br>` 태그

- 일반적으로 enter 를 사용하는 줄바꿈이 적용 되지 않기 때문에 `<br>` 태그를 사용한다
- 닫는 태그 없이 단독 사용

```html
<!DOCTYPE html>
<html lang="ko">
  <head>
    <meta charset="UTF-8" />
    <title>줄바꿈연습</title>
  </head>
  <body>
    <h1>제목</h1>
    <p>p태그 1번</p>
    <!-- p로 나누는건 -->
    <p>p태그 2번 여기에 br 태그-><br />라마바사</p>
    <h2>h2 태그</h2>
    <h2>h2 태그 2번여기에 br 태그-><br />있다</h2>
  </body>
</html>
```
